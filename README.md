## 使用 Vue.js 开发一个简单的员工数据列表：

- 每一个员工有「姓名，职位，出生日期，薪资」这四个属性
- 网页上以列表或者表格的方式来输出这些员工
- 可以添加 / 删除 员工
- 可以任意修改一个员工的任意信息
